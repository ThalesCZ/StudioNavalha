Para execução do projeto sera necessàrio ter o NodeJS instalado.

1. Execute o comando "npm install" no terminal para instalação da pasta node_modules do projeto.

2. Configuracão das variáveis de ambiente. Utilize o .envteste para formular o seu .env (renomear para .env após configuracão)

4.  Execute o comando "npm test" no terminal para a realização dos testes unitários.
